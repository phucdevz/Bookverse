## 📖 Giới thiệu
**Bookverse** là nền tảng thương mại điện tử chuyên về sách, cho phép người dùng **tìm kiếm, mua và quản lý sách trực tuyến**.  
Dự án hướng tới **bảo mật, dễ mở rộng và trải nghiệm người dùng tối ưu**.

---

## ✨ Chức năng nổi bật
- 🔍 Tìm kiếm sách theo tên, tác giả, ISBN  
- 👤 Quản lý tài khoản: đăng ký, đăng nhập, hồ sơ cá nhân  
- 🛒 Giỏ hàng & thanh toán nhiều phương thức  
- 📦 Quản lý đơn hàng cho khách hàng & admin  
- ⭐ Đánh giá, nhận xét, wishlist  
- 📱 Giao diện responsive (thiết kế từ Figma)  
- 📧 Email thông báo: xác nhận đơn, bản tin  

---

## 🛠️ Công nghệ
- 🎨 **Frontend:** PHP, HTML, CSS  
- ⚙️ **Backend:** Node.js (Express.js)  
- 🗄️ **Database:** MongoDB  
- 🖌️ **UI/UX:** Figma  
- 🔗 **Version Control:** Git + GitHub

## ⚡ Cài đặt & Chạy thử

### Backend
```bash
git clone https://github.com/your-username/bookverse-backend.git
cd bookverse-backend
npm install
npm run dev
```
### Frontend

```bash
git clone https://github.com/your-username/bookverse-frontend.git
cd bookverse-frontend
php -S localhost:8000
```
